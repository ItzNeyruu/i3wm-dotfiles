# i3wm-dotfiles
Simple i3wm Ricing Dotfiles

# things needed for dotfiles:
- `i3`
- `i3lock-color` - Advanced Customization
- `nitrogen` - Wallpaper
- `polybar` - Bar
- `rofi` Applications Menu, Disk Menu, and Power Menu
- `picom` Compositor for X11

- `JetBrainsMono Nerd Font`
